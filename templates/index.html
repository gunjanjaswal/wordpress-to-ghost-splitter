Upload your WordPress XML export file to begin

Maximum file size: 500MB
How to export your WordPress content
  1. Log in to your WordPress admin dashboard
  2. Go to Tools > Export
  3. Select All content (or choose specific content types)
  4. Click Download Export File
  5. Upload the downloaded XML file here
Why do I need this tool?

Ghost has a limitation on the number of posts that can be imported at once. If you have a large WordPress site (like your 3000+ posts), you'll get the error: "Your file has too many posts, try a file with less posts".

This tool splits your WordPress XML export into smaller chunks that Ghost can handle.
